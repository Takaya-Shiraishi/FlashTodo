# Flash Todo
シンプルなToDo管理アプリです。

## 特徴
- localStorageでクッキーにTodoリストを保存します。
- ダークテーマに対応しています。

## 技術要件
|使用したもの|使用技術|
|:-:|:-:|
|環境|vite|
|ライブラリ|React|
|使用言語|TypeScript|
|CSS|Bootstrap|
|CI/CD|GitHub Actions|
|デプロイ|GitHub Pages|
